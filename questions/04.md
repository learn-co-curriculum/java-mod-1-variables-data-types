# Matching Question

Match the term with the correct description.

## Correct

- `boolean`
- Stores true or false values.

## Correct

- `int`
- Stores whole numbers.

## Correct

- `double`
- Stores fractional numbers.

## Correct

- `char`
- Stores a single character or ASCII value.
