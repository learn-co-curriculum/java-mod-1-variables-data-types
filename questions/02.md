# Short Answer Question

What is the output of the following program?

```java
public class Main {
    public static void main(String[] args) {
        String cat = "Grumpy Cat";
        cat = "Tardar Sauce";
        System.out.println(cat);
    }
}
```

## Correct

Tardar Sauce

## Incorrect

> This program would print out "Tardar Sauce".
